PARKIRKU - LANDING PAGE SISTEM PARKIR DIGITAL KAMPUS

Cara menjalankan:
1. Ekstrak file ZIP.
2. Buka folder parkir-digital-kampus-final.
3. Klik dua kali index.html.

Atau lewat Laragon:
1. Ekstrak folder ke C:/laragon/www/
2. Buka browser.
3. Akses http://localhost/parkir-digital-kampus-final/

Struktur halaman:
- index.html = Beranda / Landing Page
- pages/fitur.html = Halaman Fitur
- pages/alur.html = Halaman Alur Sistem
- pages/statistik.html = Halaman Statistik
- pages/kontak.html = Halaman Kontak
- pages/login.html = Halaman Login Dummy

Catatan:
- Project ini masih statis.
- Belum terhubung database.
- Belum ada backend PHP.
- Cocok untuk tahap awal sebelum database MySQL/MariaDB dibuat di Laragon.
